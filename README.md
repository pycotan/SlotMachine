# Slot Machine Game

JavaScriptで作成した、シンプルで直感的に遊べるスロットマシンアプリです。

## 🚀 デモ（実際に遊ぶ）
こちらのURLからブラウザ上で今すぐプレイできます：
[https://pycotan.github.io/SlotMachine/](https://pycotan.github.io/SlotMachine/)

## ✨ 特徴
- **シンプル操作:** スタートボタンを押してリールを止め、絵柄を揃える王道の楽しさを追求しました。
- **レスポンシブ対応:** PCでもスマートフォンでも快適に動作します。

## 🛠 技術スタック
- **Language:** JavaScript, HTML5, CSS3
